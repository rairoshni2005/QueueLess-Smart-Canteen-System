# 📱 QueueLess Project Overview

## What is QueueLess?

QueueLess is a campus-focused food ordering and queue management system that helps students order ahead, track order progress in real time, and avoid cafeteria wait times. It also gives campus food vendors tools to manage menu items, monitor orders, view analytics, and use AI-assisted demand prediction.

## Core Value

QueueLess solves two major campus pain points:

- For students: long queues, unpredictable wait times, and poor visibility into cafeteria availability.
- For vendors: demand forecasting, inventory control, order management, and customer feedback.

## Key Components

- **Frontend:** React.js app built with Vite, TailwindCSS, and Socket.io client.
- **Backend:** Express.js API using Mongoose to connect to MongoDB.
- **Database:** MongoDB Atlas or local MongoDB via `MONGODB_URI`.
- **AI Service:** Python FastAPI service providing demand predictions.

## Target Users

- **Students** who want faster, modern cafeteria ordering and queue tracking.
- **Vendors** who need real-time order management, inventory visibility, and analytics.
- **Campus operations teams** who want better cafeteria planning and demand forecasting.

## Why it matters

QueueLess turns the cafeteria experience from a waiting game into a predictable, digital process. Students save time and vendors make more accurate inventory and staffing decisions.
